# Nerimity-docker

## Requirements

Windows:
- Windows Subsystem for Linux
- Docker inside WSL (see <https://docs.docker.com/desktop/wsl>)

Linux:
- Docker (see <https://docs.docker.com/engine/install> for your own distro)

## How to run?

### Windows:
```ps1
wsl.exe
```

```sh
./nerimity-docker.sh up
```

### Linux:
```sh
./nerimity-docker.sh up
```
